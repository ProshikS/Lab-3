## Задача 1

### Текст задачи
Решить задачу из лабораторной работы №2 (Задание 1. List.map) для 
последовательности 

(Получить список из сумм цифр натуральных чисел, содержащихся в исходном 
списке )

### Алгоритм решения

пользователь вводит количество элементов
если пользователь ввел значение меньшее нуля или строку, то происходит повторный запрос
запускается фукнция NaturalNumbers и пользователь вводит натуральные числа
если пользователь вводит ненатуральное число или строку, то происходит повторный запрос
затем, с помощью метода Seq.map формируется новый список, где элементу соответствует сумма цифр числа
Затем вывод результата

# Задание 2

## Задача 1

### Текст задачи
Решить задачу из лабораторной работы №2 (Задание 2. List.fold) для 
последовательности 
(Найти сумму тех элементов списка, которые начинаются на заданную цифру.)

### Алгоритм решения

вызывается функция InputAmount где пользователь вводит количество элементов списка
если пользователь вводит отрицательное число или строку, то происходит повторный запрос ввода числа
вызывается функция InputNumber где пользователь вводит элементы
если пользователь вводит строку, то происходит повторный запрос ввода
вызывается функция InputDig где пользователь вводит нужную цифру
если пользователь вводит число, либо цифру меньшую нуля, либо строку, то происходит повторный запрос ввода
затем вычисляется сумма чисел начинающихся на цифру dig и программа выводит ее на экран
